# Redes de Computadores (T02) 🌐

Repositório dedicado aos materiais e estudos da disciplina de Redes de Computadores do **Inatel**.

## 📂 Conteúdo
* **Teoria:** Notas de aula e resumos 
* **Exercícios:** Resolução das listas propostas
* **Laboratório:** Relatórios e arquivos de configuração das práticas
* **Estudo:** Flashcards (Anki) e material de apoio (Livro indicado)

## 🛠️ Ferramentas Utilizadas
* **VM VirtualBox:** Utilizada para rodar o Debian em laboratório e realizar testes de conexões NAT e Bridge
* **Debian:** Sistema operacional utilizado nos laboratórios
* **Anki:** Utilizado para revisão ativa através de flashcards
